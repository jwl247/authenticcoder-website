Phoenix Package Handler

Universal intake • Automatic tracking • Opt-in peer-reviewed distribution

One pipeline to rule them all

Download anything — script, package, config, binary — and Phoenix instantly registers it with a deterministic hex ID, sidecar metadata, clonepool versioning, and custody log. No more hunting through npm, pip, apt, cargo, etc.

Why Phoenix?

  • Intercepts every file that enters your system
  • Creates consistent hex identities + sidecars
  • Tracks custody in SQLite + Cloudflare D1
  • Opt-in peer review before anything is widely distributed
  • Works across Linux, macOS, Windows

Quick Start

Run the installer, then use intake ./mytool.sh or intake backend nodejs. Everything becomes instantly trackable.
